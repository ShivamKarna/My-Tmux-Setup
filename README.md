# TMUX Config

A personal tmux configuration optimized for productivity, mouse support, easy pane resizing, and Vim-style navigation.

## Features

- Mouse support for easier pane selection and scrolling  
- Vi-style keybindings in copy mode  
- Horizontal and vertical split shortcuts (`|` and `-`)  
- Quick reload of config (`r`)  
- Pane resizing with `hjkl` keys  
- Toggle zoom for a pane (`m`)  
- Integrated tmux plugins via [TPM](https://github.com/tmux-plugins/tpm):  
  - `vim-tmux-navigator` – seamless navigation between tmux panes and Vim/Neovim  
  - `tmux-themepack` – easy theme management  
- Resurrect pane contents with `tmux-resurrect`  
- Powerline-style theme by default (`cyan`)

# Keybindings

| – horizontal split

– vertical split

r – reload tmux config

Ctrl-h/j/k/l – resize panes

m – toggle pane zoom

v – begin selection in copy mode

y – copy selection
